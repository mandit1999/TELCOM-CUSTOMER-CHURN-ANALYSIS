# TELCOM-CUSTOMER-CHURN-ANALYSIS

# Project Description
Customer churn is a major challenge in the telecom industry. This project analyzes telecom customer data to identify patterns and key factors that influence customer churn. The insights from this analysis can help telecom companies improve customer retention and reduce revenue loss.
# Project Objectives
Analyze customer demographics, services, and billing details
Identify factors responsible for customer churn
Visualize churn trends and patterns
Provide actionable business insights for customer retention
# Dataset Information
Dataset Name: Telecom Customer Churn Dataset
Type: Structured CSV data
Target Variable: Churn (Yes / No)
Features Include:
Customer demographics (Gender, Senior Citizen, Partner, Dependents)
Services (Internet Service, Online Security, Tech Support, Streaming)
Account details (Tenure, Contract Type, Monthly Charges, Total Charges)

# Key Insights
Customers with month-to-month contracts show higher churn
New customers (low tenure) are more likely to churn
Higher monthly charges increase churn probability
Customers without value-added services (Online Security, Tech Support) churn more
# Visualizations Included
Overall churn distribution
Churn by contract type
Tenure vs churn analysis
Monthly charges vs churn
Service usage impact on churn
(All charts are available in the notebook / Power BI dashboard)
# Conclusion
The analysis shows that contract type, tenure, pricing, and service usage are major drivers of customer churn. Telecom companies can reduce churn by encouraging long-term contracts, improving onboarding for new customers, and promoting value-added services.
